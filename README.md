# Xenoglossia Frontend

Frontend do projeto open source de tradução, desenvolvido com TypeScript, React, Vite e Tailwind CSS.

## Instalação

1. Clone o repositório:

```bash
git clone https://github.com/Gabrielmedeiros1999/frontend_xenoglossia
cd seu-repositorio
```

2. Instale as dependências:
 ```bash
npm i
 ```

3. Criar um arquivo .env
```env
VITE_API_URL=...
 ```

4.Executando a aplicação
 ```bash
npm run dev
 ```
